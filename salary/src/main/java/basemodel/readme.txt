薪水支付案例：
            实现：为每个雇员支付薪水。按照规定方法准时给雇员支付正确数目的薪水，同时必须从雇员薪水中减去各种扣款。

            雇员分类：
                    1.雇员为钟点工，按照雇员记录中每小时报酬字段值进行支付。临时雇员每天提交工作时间卡（记录了日期以及工作和时间）
                                    如果每天工作时间超过8小时，超过时间按照正常工资1.5倍支付。每周五支付

                    2.正常雇员：月底对他们进行支付

                    3.带薪雇员：根据销售情况，进行支付酬金，需要他们提交销售凭条，记录销售日期和数量

            支付方式：
                    雇员可以选择支付方式：
                                         1.将支付支票邮寄到指定的邮政地址

                                         2.支票保存在出纳人员那里随时提取

                                         3.将薪水直接存入他们的银行账号

            扣除情况：
                   协会雇员，雇员记录中有一个每周应付款项字段，需要从薪水中扣除，有时也会针对单个雇员进行扣除。协会每周会提交
                             服务费用，服务费用会从相应雇员下个月薪水总额中扣除

            工作方式：
                     每天运作一次，为当天相应雇员进行支付，系统会告知雇员的支付日期，这样会计算从上次支付日期到这次支付日期印支付
                     的费用


用例：1.增加雇员
      2.删除雇员
      3.登记时间卡
      4.登记销售卡
      5.登记协会服务费
      6.更改雇员明细
      7.运行支付系统
